PyPI Objects Reference
======================

This module contains objects returned by :class:`pypiwrap.client.PyPIClient` . They represent the data provided by the `JSON API <https://docs.pypi.org/api/json/>`_ and the `Stats API <https://docs.pypi.org/api/stats/>`_.

.. warning::
   Some aspects of the documentation have been inferred from source code and issues of the Warehouse software powering PyPI. While this API seems to have remained stable, changes may occur.
