Simple Repository Objects Reference
===================================

This module contains objects for the Simple Repository API (also known as the Index API). These objects are returned by :class:`pypiwrap.client.SimpleRepoClient`.

Note that pypiwrap implements the JSON-based API specified in PEP 691 rather than the HTML-based API.
