Simple Objects Reference#

These objects pertain to the Simple Repository API and are used by pypiwrap.SimpleClient.

class pypiwrap.objects.simple.DistributionFile(filename: str, url: str, size: Size, hashes: dict[str, str], upload_time: datetime | None = None, requires_python: str | None = None, core_metadata: bool | dict[str, str] | None = None, dist_info_metadata: bool | dict[str, str] | None = None, has_sig: bool | None = None, yanked: bool | str | None = None)#

Bases: APIObject

A file for a package distribution

core_metadata: bool | dict[str, str] | None = None#

  • If a boolean, whether this file has associated metadata.

  • If a dictionary, a mapping of hashes to encoded metadata file hashes.

dist_info_metadata: bool | dict[str, str] | None = None#

Contains the same values as DistributionFile.core_metadata. Where available, prefer the use of core_metadata over this attribute.

filename: str#

The filename of the distribution

has_sig: bool | None = None#

Whether a GPG signature is included with the file

hashes: dict[str, str]#

A mapping of hashes for this file. Similar to ReleaseFile.digests.

requires_python: str | None = None#

The version constraints for this file if specified

size: Size#

The size of the distribution

upload_time: datetime | None = None#

The upload time for this file

url: str#

The download URL for the file

yanked: bool | str | None = None#

  • If a boolean, whether the file was yanked.

  • If a string, why the package was yanked.
